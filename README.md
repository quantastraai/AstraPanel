# 🚀 AstraPanel – Finance Analytics Dashboard

A modern and premium finance analytics dashboard built with a dark glassmorphism-inspired design. AstraPanel provides a clean and interactive interface for monitoring income, expenses, savings, transactions, and financial insights in a visually rich way.



## ✨ Features

- 📊 Interactive Analytics Dashboard
- 💰 Income, Expenses & Savings KPI Cards
- 💳 Realistic Banking Card UI
- 📈 Monthly Analytics Chart
- 📉 Credit Score Tracking
- 🔔 Notification Badge System
- 📋 Recent Transactions Section
- 🥧 Statistics Overview Chart
- 🎨 Dark Premium UI Theme
- ⚡ Smooth Hover & Micro Interactions
- 📱 Responsive Layout
- 🧩 Collapsible Sidebar Navigation
